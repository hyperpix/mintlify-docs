# Technology Stack

# Core
- **Language:** TypeScript
- **Frontend Framework:** Next.js 16 (App Router)
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4 & Radix UI (shadcn/ui)

# Backend & Data
- **Platform:** Supabase (Auth, Database, Storage)
- **Database:** PostgreSQL (via Supabase)
- **ORM/Client:** Supabase SDK & Prisma (found in backup)

# Specialized Libraries
- **Animations:** Framer Motion & React Three Fiber (Three.js)
- **Data Visualization:** Recharts
- **PDF Generation:** @react-pdf/renderer & jspdf
- **Email:** Resend
- **State Management:** React Server Actions & Hooks

# Infrastructure
- **Package Manager:** pnpm
- **Development Tools:** ESLint, Prettier
