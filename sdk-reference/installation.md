---
title: "Installation"
description: "How to install the Montra SDK"
---

The `@montra/sdk` is a unified package that works in Node.js, the browser, and across all major JavaScript frameworks.

### NPM

```bash
npm install @montra/sdk
```

### Yarn

```bash
yarn add @montra/sdk
```

### PNPM

```bash
pnpm add @montra/sdk
```

## Requirements

- Node.js 18.0.0 or later
- React 18+ (for React/Next.js components)
